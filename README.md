# BrainF Programs
A Collection of BrainF Programs.

# Running a Program
`./bfc/bfc [programName]`

Example: `./bfc/bfc helloWorld.bf`
```
[bfc] BFC - A BrainF Compiler
[bfc] Loading ../helloWorld.bf
[bfc] Parsing.   . 99%
[bfc] Done parsing.
[bfc] Compiling...
[bfc] Done Compiling.
[bfc] Executing.
Hello World!
[bfc] Execution complete.
```
